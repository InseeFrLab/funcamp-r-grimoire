fmt1Decimale

Description

Formate un nombre

Usage

fmt1Decimale(x)

Arguments

x

un nombre

Details

La fonction formate avec une decimale, la virgule pour distinguer unites et decimales, l'espace comme separateur de milliers.

Value

Le nombre formate

Emploie

Les fonctions fmt et https://www.rdocumentation.org/packages/base/versions/3.6.2/topics/format

Warning

N'utilise pas de notation scientifique

Author(s)

M. Martin, martin@insee.fr
