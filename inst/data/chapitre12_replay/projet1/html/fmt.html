fmt

Description

La fonction formate un nombre avec le nombre de decimales specifié, la virgule pour distinguer unités et décimales, l'espace comme séparateur de milliers.

Usage

fmt(x, nombre_decimales)

Arguments

x

la quantité à formater

decimales

le nombre de decimales

Examples

fmt(100, 2)
fmt(1234.56789, 2)
