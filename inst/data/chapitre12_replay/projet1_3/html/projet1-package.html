projet1 : un package pour formater des nombres

Description

Le package "projet1" fournit deux fonctions : fmt et fmt1Decimale.

Les fonctions du package "projet1"

utilisent la virgule au lieu du point pour les nombres decimaux, et l'espace " " pour separer les milliers.
